What?

aliv is a light, fast, powerful and intuitive one liner live-reloading Node.js server. Created to help you get stuff done.

You can choose the way to work with aliv: cli, .alivrc or a local node module!

Why?

Created to push the web forward.

It should be easy to run a HTTPS server.

It should be easy to proxy a request to any server.

It should serve the content fast.

This is what aliv aims to solve.

aliv simplifies a lot of headache we have when developing complex web apps. Proxy request/responses, implementing HTTPS, refreshing all your browsers with each change, serving content gzipped, having the server not consuming the whole memory/cpu, etc - was never so easy!

How?

After installing it, go to the folder with the index.html and run $ aliv (if you installed it globally) or $ npm run your-script-here if you installed it locally.

Either use the options inline, by calling $ aliv --port 9876 --quiet

Or create a file called .alivrc with the following content:

{
      "quiet": true,
      "port": "9876"
}
